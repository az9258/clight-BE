# 请求用户信息

1. 请求信息
   - 请求方法: `GET`
   - 请求路径: `/api/auth/currentUser`
   - 请求头: Authorization

2. 请求体: `Bear + " " + token`

2. 响应信息

   - 成功响应示例

   ```json
   {
        "success": true,
        "code": 200,
        "msg": "操作成功",
        "data": {
             "id": {
                  "timestamp": 1744384027,
                  "date": "2025-04-11T15:07:07.000+00:00"
             },
             "username": "test001",
             "email": "123456@gmail.com",
             "roles": [
                  "NORMAL"
             ],
             "status": 1
        }
   }
   ```

   - 失败响应示例

   ```json
   {
        "success": false,
        "code": 10003,
        "msg": "token 不合法",
        "data": null
   }
   ```



# 退出登入

1. 请求信息
   - 请求方法: `GET`
   - 请求路径: `/api/logout`
   - 请求头: `Authorization`

2. 请求体: `Bearer + " " + token`

3. 响应信息

   - 成功响应示例

   ```json
   {
        "success": true,
        "code": 200,
        "msg": "操作成功",
        "data": null
   }
   ```

   - 失败响应示例

   ```json
   {
        "success": false,
        "code": 10003,
        "msg": "token 不合法",
        "data": null
   }
   ```
